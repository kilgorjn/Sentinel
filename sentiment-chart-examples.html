⚛️ Net Force Chart (Weighted Sentiment Momentum)

What this answers: Is sentiment predominantly positive or negative, weighted by event importance (HIGH=3x, MEDIUM=2x, LOW=1x)?

Positive Force (Bullish)
Negative Force (Bearish)
Event Volume (Mass)
Left axis (bars): Total weighted event volume
Example: 1 HIGH + 2 MEDIUM + 3 LOW = (1×3)+(2×2)+(3×1) = 10 mass units

Right axis (line): Net directional pressure from -1 to +1
• +1 = all positive sentiment
• 0 = balanced (equal positive/negative weighted events)
• -1 = all negative sentiment

Line color: Green when >+0.05 (bullish), Red when <-0.05 (bearish)
Steepness: Faster swings = high velocity (sentiment changing rapidly)

✓ Correlates with trading activity spikes | ✓ Weights HIGH events heavily | ✓ Shows sentiment direction

Recommendations

For Dashboard Overview: Stacked Bar Chart
• Shows both volume AND composition in one chart
• Easy to compare across hours (each bar is a time period)
• Visually different from the line chart above it
• Matches the dark theme and existing design language
• Sentiment proportions are immediately visible

For Financial Momentum: Net Force Chart ⚛️
• Answers: "Is sentiment bullish or bearish, weighted by importance?"
• Background bars = total weighted event volume (mass)
• Foreground line = net sentiment direction (-1 to +1)
• HIGH-classified events count 3x, MEDIUM 2x, LOW 1x
• Steep line = sentiment swinging fast (high velocity)
• Useful for correlation: when Net Force spikes, check if login volume also spiked in Splunk

How to read it:
• Net Force = +0.8 → strongly positive sentiment (bullish), expected buying
• Net Force = -0.7 → strongly negative sentiment (bearish), expected selling
• Net Force = 0 → balanced sentiment, no clear direction
• Steep upward slope → sentiment rapidly improving (momentum building)
• Steep downward slope → sentiment rapidly deteriorating (selling pressure building)
